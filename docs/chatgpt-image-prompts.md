# ChatGPT Image Prompts

В текущей реализации используются CSS/SVG/React-мокапы без внешних изображений. Ниже — готовые промпты для генерации дополнительных визуальных материалов через ChatGPT Image.

## 1. Main dashboard preview

Create a realistic dark SaaS analytics dashboard for a fictional platform called EventPulse. The interface should show live event streams, metrics cards, platform activity charts, API status widgets, and webhook logs. Use a black graphite background, subtle green neon accents, thin borders, and realistic product UI details. It must not look like a web agency landing page. No real company logos. No real personal data.

## 2. Live events stream preview

Create a focused product UI screenshot for EventPulse showing a live event stream table. Include event names, anonymized user IDs, timestamps, tags, filters, and realtime status indicators. Use a dark graphite SaaS interface with green accent highlights, compact typography, thin separators, and a professional analytics product feel. No real company logos. No real personal data.

## 3. API documentation preview

Create a realistic developer documentation UI for EventPulse. Show an API request example, SDK cards, webhook events, response status indicators, and navigation for docs. Style it as a dark developer dashboard with graphite panels, subtle green highlights, readable code blocks, and neutral SaaS branding. No real domains except fictional examples. No secrets or tokens.

## 4. Service status preview

Create a dark service health dashboard for EventPulse. Show service rows, uptime, latency, incident-free status, green health indicators, compact cards, and a calm operations interface. Use black graphite surfaces, thin borders, restrained green neon accents, and realistic product details. No real infrastructure details, IP addresses, or secrets.

## 5. Abstract event data visualization

Create an abstract but realistic visualization of event analytics data for EventPulse: flowing event timelines, small charts, heatmap cells, and telemetry streams inside a dark SaaS dashboard environment. Use black graphite background, green accent lines, subtle cyan secondary highlights, and precise UI geometry. No real company logos, no real personal data, no generic stock-photo look.
